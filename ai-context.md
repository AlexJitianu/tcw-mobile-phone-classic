# Mobile phone project context


#Templates
The templates are located in `templates` directory of the project directory.

The name that you use to save the template will be the name that appears in the new document wizard, including capitalization, space, and characters (for example, My Custom Template1.xml will appear in the new file wizard as My Custom Template1).
